Included Tools

One BenchKit Repo, Two Focused Applications

PlateMaster

Well Plate Designer

Generate structured plate maps using treatment groups, time points, biological replicates, and technical replicates. Export the full plate board as PNG, CSV, or protocol text.

  • 6, 12, 24, 48, 96, and 384-well support
  • Color-coded group assignment with control column reservation
  • Overflow reporting when requests exceed plate capacity
Launch PlateMaster
Hydrogel

Dual-Syringe Batch Calculator

Calculate GelMA, fibrinogen, thrombin, and top-up volumes in real time for two-syringe hydrogel preparation with clean error handling and exportable recipe panels.

  • Real-time C1V1 = C2V2 calculation flow
  • Separate Syringe A and Syringe B recipe tables
  • Copyable summary and PNG export for recipes
